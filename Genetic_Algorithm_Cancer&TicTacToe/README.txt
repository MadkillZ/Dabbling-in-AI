To run you simply need to click on the run.bat file.
Please make sure to give valid inputs (mostly numbers), otherwise the program probably wont run

If your pc doesnt have javac you will probably need java 16, since that was what it was compiled with.

This is a genetic algorthm that evolves making use of mutation and crossover of fittest parents. This makes use of binary trees and randomly generates according to a seed.

This genetic algorithm was made for inputs of 9, and the last field of the data is used to determine what the outcome should be or if the algorithm was correct when testing. In the breast cancer part the 1 means the person has cancer, and 0 means they dont. Tick-tac-toe is the same except the last value is which person will win.
